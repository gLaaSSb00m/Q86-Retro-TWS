# Q86 Retro Wireless In-Ear Headset Website

A modern, responsive landing page for the Q86 Retro Wireless In-Ear Headset by Sound by KFG. This website showcases the product's retro design, premium features, and provides an easy ordering system.

## Project Overview

This is a static website built to promote and sell the Q86 Retro Wireless In-Ear Headset. The site features a clean, modern design inspired by vintage cameras and classic audio equipment, combined with cutting-edge wireless technology.

### Key Features
- Responsive design that works on all devices
- Interactive product gallery with image thumbnails
- Color selection options
- Comprehensive features showcase
- Integrated order form with validation
- WhatsApp integration for customer support
- Admin panel for order management
- Google Sheets integration for order tracking

## Technologies Used

- **HTML5**: Semantic markup and structure
- **CSS3**: Modern styling with CSS Grid, Flexbox, and custom properties
- **JavaScript (ES6+)**: Interactive functionality and form handling
- **Font Awesome**: Icons for enhanced UI
- **Google Fonts**: Montserrat and Orbitron fonts

## Setup Instructions

Since this is a static website, no server setup is required. Simply open the `Home.html` file in any modern web browser to view the website.

### Local Development
1. Clone or download the project files
2. Navigate to the project directory
3. Open `Home.html` in your preferred web browser
4. The website will load with all styling and functionality

- Chrome (recommended)
